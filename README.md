# Amigo Secreto

Este projeto simula um sorteio de amigo secreto usando lógica de programação com JavaScript.

##  Como executar

1. Clone o repositório:
2. Abra o arquivo `amigo_secreto.js` em seu editor.
3. Execute com Node.js ou em um navegador.
